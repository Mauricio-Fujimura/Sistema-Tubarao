Esse projeto sera um sistema de vendas, controle de estoque e caixa simples para uso pessoal
vai conter a opção de visualizar e editar estoque, e adicionar produtos ao estoque
na parte de vendas obviamente a de vender, adicionar desconto, informar troco, informar a forma de pagamento e registrar a venda
no controle de caixa vão ter as opções de adicionar a quantidade de dinheiro do dia, de sangria e fechamento de caixa
